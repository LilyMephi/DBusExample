###Для Загрузки
```bash
   git clone https://github.com/LilyMephi/DBusExample/
   cd DBusExample
   qmake project.pro
   make
   qmake client.pro
   make
```
#Измените в файле `client.pro` 
`/path/to/lib` на путь к библиотеке SharingService
#Для запуска примера нужно скачать библиотеку SharingService
