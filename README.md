###Для Загрузки
```bash
   git clone https://github.com/LilyMephi/DBusExample/
   cd DBusExample
   qmake project.pro
   make
   qmake client.pro
   make
```
# Для запуска в разных терминалах нужно запустить
```bash
  ./ClientService
  ./SharingService
```
